# Schadensprotokoll - ALMAS INDUSTRIES

Digitales Protokoll zur Schadensmeldung am Firmenfahrzeug.

## Features

- **Allgemeine Angaben**: Datum, Uhrzeit, Ort, Fahrer
- **Fahrzeugdaten**: Kennzeichen, Modell, Kilometerstand
- **Schadenshergang**: Art des Verschuldens, Polizei, Schuldfrage
- **Schadenserkl√§rung**: Unfallskizze, Schadensfotos
- **Pflichtfotos**: Front, Heck, Seiten, Schadensdetails, Tacho
- **Fahrzeugzustand**: Fahrbereit Ja/Nein, Standort mit Google Maps Validierung
- **Bildkompression**: Automatisch max. 1600px, JPEG 75%
- **Dark/Light Mode**
- **PDF-Druckansicht**
